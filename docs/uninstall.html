Uninstalling Retry

If you wish to uninstall the Retry System, and you're not using a tool like Callisto that can rebuild the rom from scratch easily, it is easy to do following these steps:

  • Delete the retry.asm file from UberASM Tool's library folder.
  • Delete the retry_gm.asm files from UberASM Tool's gamemode folder.
  • Delete the retry_config folder from UberASM Tool's top folder.
  • In UberASM Tool's list.txt file, remove the line * retry_gm.asm under gamemode: (or just remove ,retry_gm.asm if you have multiple files after *).
  • Run UberASM Tool.
  • Patch the retry_unpatch.asm file found in the src folder using Asar.
