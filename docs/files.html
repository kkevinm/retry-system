Graphics

The graphic files used by Retry can be found in the retry_config/gfx folder.

FilePurpose
letters1.binLetters used in the prompt when !no_prompt_box = 0
letters2.binLetters used in the prompt when !no_prompt_box = 1
coin.binTwo 8x8 tiles used by the sprite status bar coin counters (the first is used by the coin counter, the second by the Dragon Coin counter)
digits.binNumbers used by the sprite status bar timer and coin counter
indicator.binSingle 8x8 tile used by the retry indicator
item_box_8x8.binSingle 8x8 tile used to draw the smaller Item Box
item_box_16x16.binFour 8x8 tiles used to draw the full Item Box
timer.binSingle 8x8 tile used for the sprite status bar timer icon
