Known Issues and Limitations

This page lists known issues and limitations of the Retry System. Note that, unless stated otherwise, all of these are not possible or planned to be fixed.

  • Dying after the goal tape ("Kaizo trap") will cause the circle effect to disappear if the prompt shows up and the prompt black background is not disabled.
  • When the prompt appears (with the black background enabled), it will mess up level effects that use windowing (for example, Fading Lights UberASM, or the vanilla spotlight effect).
  • The sprite status bar will appear behind layer 3 with top priority enabled.
  • If the prompt black background is disabled, the prompt text will appear behind layer 3 with top priority enabled.
  • The sprite status bar will appear behind layer 2 in level modes where layer 2 has higher priority than sprites (like 1F).
  • If the prompt black background is disabled, the prompt text will appear behind layer 2 in level modes where layer 2 has higher priority than sprites (like 1F).
  • Some tiles in the sprite status bar may appear behind other sprites (mostly projectiles and stuff like that).
  • Some graphics in vanilla mode 7 bosses (like Roy's candles or Morton's pillars) will disappear when the prompt appears or when the sprite status bar is enabled.
